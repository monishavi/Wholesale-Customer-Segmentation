# Wholesale Customer Segmentation using K-Means Clustering

## Overview
This project performs customer segmentation using the K-Means Clustering algorithm.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Dataset
Wholesale Customers Dataset (440 records)

## Machine Learning Algorithm
- K-Means Clustering

## Evaluation
Silhouette Score: 0.548

## Customer Segments
- Cluster 0: Grocery-Oriented Customers
- Cluster 1: Regular Customers
- Cluster 2: Premium Customers
